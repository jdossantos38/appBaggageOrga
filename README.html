🧳 Liste de Bagages — README

Générateur intelligent de listes de bagages pour vos voyages

📖 Description

Liste de Bagages est une application web progressive (PWA) qui génère automatiquement des listes de bagages personnalisées selon vos conditions de voyage : saison, destination, transport, hébergement, durée et participants (bébé, enfant, ado, adulte).

L'application fonctionne entièrement dans le navigateur, sans serveur ni connexion requise après le premier chargement. Toutes les données restent sur votre appareil (localStorage).

✨ Fonctionnalités

Génération intelligente

  • Liste adaptée selon la saison (été, hiver, printemps, automne)
  • Objets ajustés selon la destination (France, Europe UE, hors Europe)
  • Contenu filtré par transport (voiture, avion, train, bateau)
  • Objets par hébergement (hôtel, Airbnb, camping, chez des amis)
  • Quantités calculées selon la durée du voyage
  • Onglets séparés : objets communs + un onglet par participant
  • Catégories par tranche d'âge : bébé (0-2), enfant (3-12), ado (13-17), adulte (18+)
  • Smartphone + chargeur ajouté automatiquement pour les enfants de 12 ans et plus
  • Activités spéciales : randonnée, ski, baignade, fête, vélo

Personnalisation de la liste

  • Cocher les objets au fur et à mesure — progression sauvegardée automatiquement
  • Ajuster les quantités avec les boutons −/+
  • Supprimer des objets prédéfinis avec le bouton ×
  • Ajouter des objets personnalisés en bas de chaque catégorie
  • Réordonner les objets par glisser-déposer
  • Sauvegarder la configuration (suppressions, quantités modifiées et objets ajoutés)
  • Option « Appliquer à toutes les configurations » pour garder les modifications d'un voyage à l'autre

Sauvegarde et export

  • Profils de participants — sauvegarder/charger les noms et âges
  • Voyages complets — sauvegarder/charger toute la configuration du formulaire
  • Copier la liste en texte (presse-papier) pour partager par message
  • Imprimer la liste complète

Autres fonctionnalités

  • 🏠 Page d'accueil avec la liste de vos voyages enregistrés — cliquez pour charger directement la liste
  • ⚖️ Estimation du poids total avec avertissement pour les bagages cabine (10 kg)
  • 🌐 Interface bilingue français / anglais
  • 📱 PWA — installable sur l'écran d'accueil (iOS + Android)
  • 📶 Fonctionne hors-ligne après le premier chargement
  • ❓ Aide intégrée — bouton ? flottant avec guide complet
  • 👆 Glisser-déposer tactile pour réordonner les objets sur mobile (Android + iOS)

🚀 Installation / Déploiement

  1. Copiez tous les fichiers dans un même dossier
  2. Servez avec n'importe quel serveur HTTP :
    python -m http.server 8080
  3. Ouvrez http://localhost:8080 dans votre navigateur
  4. Sur mobile : utilisez l'adresse IP de votre ordinateur (ex: http://172.20.10.3:8080)

Aucune build ni dépendance n'est requise. Tout fonctionne avec des fichiers statiques.

📲 Installation PWA

  • iPhone / iPad : Safari → Partager → « Sur l'écran d'accueil »
  • Android : Chrome → menu ⋮ → « Installer l'application »
  • PC : Chrome → barre d'adresse → icône d'installation

🔐 Confidentialité

Aucune donnée n'est envoyée à un serveur. Toutes les informations sont stockées localement dans le navigateur via localStorage. Le Service Worker met en cache les fichiers pour le fonctionnement hors-ligne uniquement.

🧳 Trip Packing List — README

Smart packing list generator for your trips

📖 Description

Trip Packing List is a progressive web app (PWA) that automatically generates personalized packing lists based on your travel conditions: season, destination, transport, accommodation, duration, and participants (baby, child, teen, adult).

The app runs entirely in the browser with no server or internet connection required after the first load. All data stays on your device (localStorage).

✨ Features

Smart Generation

  • List adapted by season (summer, winter, spring, fall)
  • Items adjusted by destination (France, Europe EU, outside Europe)
  • Content filtered by transport (car, plane, train, boat)
  • Items by accommodation (hotel, Airbnb, camping, friends' place)
  • Quantities calculated based on trip duration
  • Separate tabs: shared items + one tab per participant
  • Categories by age group: baby (0-2), child (3-12), teen (13-17), adult (18+)
  • Smartphone + charger automatically added for children aged 12 and over
  • Special activities: hiking, skiing, swimming, party, biking

List Customization

  • Check items as you pack — progress saved automatically
  • Adjust quantities with −/+ buttons
  • Remove predefined items with the × button
  • Add custom items at the bottom of each category
  • Drag & drop to reorder items
  • Save configuration (removals, quantity changes, and custom items)
  • "Apply to all setups" option to keep changes across different trips

Save & Export

  • Participant profiles — save/load names and ages
  • Full trips — save/load the entire form configuration
  • Copy list as text (clipboard) for sharing via messaging
  • Print the complete list

Other Features

  • 🏠 Homepage with your saved trips list — click to load a trip directly
  • ⚖️ Weight estimate with cabin baggage warning (10 kg)
  • 🌐 Bilingual interface — French / English
  • 📱 PWA — installable on home screen (iOS + Android)
  • 📶 Works offline after first load
  • ❓ Built-in help — floating ? button with comprehensive guide
  • 👆 Touch drag & drop to reorder items on mobile (Android + iOS)
